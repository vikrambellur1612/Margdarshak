Guidelines for Margdarshak Community Participation

These are the do's and don'ts when using the forum. Please understand that there are 1000 fellow parents in the forum, so strictly follow these guidelines so that discussions in the forum are helpful to everyone.
‚úÖ Dos - What You Should Do
Join and Post in the appropriate group(s) ONLY.
Read the earlier posts before asking a question. Please search in the group if the question is already answered.
Before you post any information, please verify if it is genuine. You might be misleading 1000 fellow parents otherwise.
Use # tag when you post a message. For example, if you have a question about JEE Exam, please use tag as #JEE so that everyone gets the context and is easier to search later. Feel free to use multiple # tags as appropriate.
Remove your message that's not relevant (Sometimes people accidentally post messages that they did not intend to).
‚ùå Don'ts - What You Should Avoid
Post anything that promotes any of the training/ commercial forums.
Post any message other than what's relevant to the competitive exams.
Post Hellos, Good morning.
Post replies like - Ok, Thanks, Oh Ok, Hmm, LoL, Ok great, Thanks for the explanation/ clarification/ information, Thanks for clearing the doubt etc. Such messages keep consuming space and do not add any real value. Strictly use emojis ONLY in such cases.
Post anything that's de-meaning others in the group or their views.
Announcing personal achievements /marks/results of your ward.
Forward any clickbait type of messages giving some link.
Create opinion polls in the forum.
